Diatonicator

Music theory practice tool.

What is Diatonicator?

Diatonicator is a tool to help musicians learn music theory and practice improvisation.

What is the point of Diatonicator?

Diatonicator was built to dynamically model music theory concepts for any key or scale. You tell Diatonicator what you would like to see, and it does the work for you.

How should I use Diatonicator?

Diatonicator can be used as a study tool, or as a reference during practice sessions. Simply pick a key and a mode to get started.
